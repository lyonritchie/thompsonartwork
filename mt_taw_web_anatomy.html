ThompsonArtwork.com is a static website – it does not use a database. Instead data is stored in flat text files throughout the git repository. GitHub Pages automatically recompiles this repo to html after each edit (commit) or push from local working copy. To streamline maintenance and publishing NetlifyCMS is connected as a front-end available at https://thompsonartwork.com/admin.

Requirements

  • A GitHub user account
  • Write privileges to the site repository provided by invite from Lyon Ritchie
  • Images named word-word-another.jpg, e.g. using the dash as a separator. Include corresponding thumbnails:
    • Size: 250x250 px
    • Name: ends with _tn.ext, e.g. word-word-another_tn.ext

Content Structure

There are differences between the public organization of the site and the private organization within NetlifyCMS.

Content Editing

Hints are included with notable fields in the NetlifyCMS editor. Fields for editable content types are also listed below.

Blog

Fields
  • Title
  • Published
  • Publish Date. Required.
  • Body
  • Description - SEO: Will be used in the site meta-description tag.
  • Topics - Be sure to match the topic name used in Topics (lowercase, dashes, etc.)
Tip:

You can upload images using the Rich Text editor. Switch back to Markdown to apply styling.

Bootstrap classes can be used for formatting as shown below. Bootstrap classnames appear between brackets { …​ } and immediately follow the image tag.

Galleries

Fields
  • Title - Appears below image. Required.
  • Description - SEO + Social Sharing.
  • Image - Images uploaded to /uploads/[galleryname]. Required.
  • Published
  • Publish Date - Affects sort order in gallery. Required.
  • Tags - Options: "cover", "featured". Show image on home page (featured) or as the gallery cover image.
  • Body - Optional content will appear below the image.
Note:

Gallery images are stored separate from the default Media location.

Archives and Topics

Archive and Topic entries must exist to provide lists of posts by year and by topic.

Fields for Archives
  • Year - in YYYY format
  • Published
  • Body
Fields for Topics
  • Topic - Lowercase, use dash for space between words.
  • Published
  • Body

Pages

Fields
  • Order - Controls the position within the main navigation
  • Body
  • Description - for SEO purposes
  • Published - Only available for Shop
Note:

No two pages are alike – adding new pages from NetlifyCMS is disabled.

Everything Else

While most content can be edited via NetlifyCMS there are a few bits tucked away in the site configuration file, namely the description, which you can override by editing the index.md file at Pages > Home.

Other defaults, such as site title and author are also stored in _config.yml.
